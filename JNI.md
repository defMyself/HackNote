# JNI

Java 本地接口（JNI)

JNI提供了将Java与C/C++, 汇编等本地代码集成的方案，该规范使得在Java虚拟机内运行的Java代码能够与其他编程语言互相操作，包括创建本地代码、跟新Java对象、调用Java方法，引用Java类，捕捉和抛出异常等，也允许Java代码调用C/C++或汇编语言编写的程序和库。作为一个标准接口，它没有对底层Java虚拟机的实现施加任何限制，并具有以下特点：

1. 二进制兼容。
